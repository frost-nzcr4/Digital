github.com

устанавливает Git Bush, там прописываем для регистрации:
git config --global user.name "NAME"
git config --global user.email "EMAIL"

для инициализации папки, переходив в неё, например
cd G:/git
и инициализируем
git init

потом вставляем адрес ГИТа, например:
https://github.com/SinTiger/Digital.git
прописываем:
git remote add ИМЯ_Репозитория https://github.com/SinTiger/Digital.git

Вытащить всё из репозитория Digital из папки master:
git pull Digital master

Создать файл:
touch README.md

Добавить файл в список отслеживаемых:
git add README.md

Добавить все файлы и папки дирректории:
git add .

Закоммитить файлы:
git commit -m "ОПИСАНИЕ"

Отправить коммиты на сервер:
git push Digital master
Далее указываем логин и пароль

проверить какие файлы отслеживаются:
git status
